# Sampurasun

## Overview
This project is a custom sundanese programming language

## Requirements
- Bash
- A Unix-like environment (Linux or macOS)
- Python
- Executable permissions for `build.sh`

## Installation
Clone the repository:

```bash
git clone https://github.com/arbipink/sulang-language.git
cd sulang-language
```

## Build
```bash
bash build.sh ./example/sampurasun.sulang
```

## Run
```bash
./sampurasun
```
